Git Command Help Page

Git Merge

  • To run git merge, run the following command:

    ./gitmerge.sh {branchname}

Create a Branch

  • To create a new branch, run the following command:

    ./gitbranch.sh {branchname}

Creating a Pull Request

  1. To Create a Pull Request, Open GitHub Desktop

  2. Write a Summary and optionally write a Description.

  3. Click on the Commit to {branchname} button.

  4. Go to the Source Control section of VS Code.

  5. Click on the Sync Changes button.

  6. Go back to GitHub Desktop and click on the Preview Pull Request button.

  7. Click on the Create Pull Request button.

  8. GitHub.com will open to your request.

  9. Click on the gear in the Add Assignees section and add Nicholas Boss.

  10. Click on the Create Pull Request button.
